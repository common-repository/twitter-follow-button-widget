== Description ==

Plugin that add widgets to add twitter  follow button to your website. Multiple widget with different configurations.

== Installation ==


1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure widget
